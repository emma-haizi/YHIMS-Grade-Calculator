## Change Log

### [v0.1.0] - 2026-06-29

#### **✨ Added** 
- **학기별 과목 구성:** 1학기 및 2학기 내 다양한 과목(국어, 수학, 사회/역사, 과학, 영어, 제2외국어, 기술가정) 선택 폼 및 각 과목 별 성적 반영비율을 보여주는 페이지 구현
- **A등급 커트라인 계산기:** 중간고사 및 수행평가 점수를 기반으로 A등급 달성을 위해 수행평가 및 기말고사에 필요한 최소 점수를 역산하는 기능 추가
- **최종 성적 계산기:** 중간고사/기말고사 및 수행평가 점수를 종합하여 최종 원점수를 산출하는 기능 추가
- **실시간 결과 출력:** 각 계산 버튼 클릭 시 하단 영역에 즉각적으로 분석 결과를 시각화하는 기능 추가
- **Enter키 입력시 자동 계산:** 점수 입력 후 'Enter'키를 누르면 즉각적으로 계산이 실행되는 단축 기능 추가(마우스 클릭 없이도 결과를 확인할 수 있도록 입력폼의 UX 개선)
- **점수별 축하 메시지:** 각 원점수 별 서로 다른 축하메시지 출력 기능 추가
